# n8n Telegram AI Appointment Agent

This repository contains a self-hosted n8n workflow for an AI-powered Telegram appointment assistant.

## What this workflow does
- Handles text and voice messages
- Validates users before processing
- Uses AI to understand appointment requests
- Prevents unsafe delete or update actions

## Tech Stack
- n8n (self-hosted)
- Telegram Bot API
- OpenAI
- PostgreSQL / Supabase

## How to use
1. Import `workflow.json` into n8n
2. Set up required credentials
3. Adjust IDs and environment variables

## Notes
This workflow is shared for portfolio and educational purposes.
Sensitive data is not included.
